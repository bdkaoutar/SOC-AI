### To detect the attacks make sure to apply the following changes to ufw

sudo ufw enable
sudo ufw logging on
sudo ufw logging low
sudo chmod 644 /var/log/ufw.log

I'm using python3.11.0
### Install the requirements :
pip install -r requirements.txt
